# Análisis de anuncios de venta de coches

Esta aplicación web fue desarrollada con **Streamlit** y permite explorar un conjunto de datos de anuncios de venta de coches.

## Funcionalidades
- Visualización de la distribución del odómetro mediante un histograma interactivo.
- Visualización de la relación entre el precio y el odómetro mediante un gráfico de dispersión.
- Interfaz sencilla e interactiva usando botones.

## Tecnologías utilizadas
- Python
- Pandas
- Plotly Express
- Streamlit
